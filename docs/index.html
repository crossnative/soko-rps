Wie sieht es ohne aus?

  1. Rechenzentrum / Hardware / Server
  2. Eigenes Deployen der Anwendung
    • zeitaufwändig
    • fehleranfällig
    • fehlende Vorhersagekraft
    • hohe Downtimes
  • Eigene Updates, Betriebssysteme verwalten, Hardware reparieren
  • Netzwerk und Sicherheitsaspekte

Wie sieht es ohne aus?

  1. Rechenzentrum / Hardware / Server
  2. Eigenes Deployen der Anwendung
    • zeitaufwändig
    • fehleranfällig
    • fehlende Vorhersagekraft

  3. Hohe Time To Market
  4. Keine Lastskalierung
  5. Kosten auch bei Nicht-Gebrauch
  6. Kein Fokus allein auf Anwendung
  • Markt hat sich verändert: hohe Ansprüche an Verfügbarkeit/Robustheit
  • zusätzlich auf Grund Risiko + Zeit seltene Releases
  • Unternehmen: Ich möchte eine Sache können und diese besser als Andere machen
